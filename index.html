Tim Berners-Lee invented the World Wide Web and made the first connection with a Web Browser and a Web Server.

What is the Internet

The Internet is a Computer Network of Inter-Connected Devices Conected to the Internet by a Physical Connection. The Internet is also Powererd by Protocols. One of Them, and the one that powers the World Wide Web, is HTTP (Hyper Text Transfer Portocol), also with other Protocols, like TLS(Transport Layer Security) for security, TCP(Transmission Control Protocol) for Relliable Packet Transmition, and IP(Internet Portocol) for Routing. They don't only Power The World Wide Web, Except for HTTP. These are not the only Protocols that power the Internet, but the most popular Protocols.

Check this course at Khan Academy if you want to learn more about the Internet or go to the Wikipedia Article.

What is HTTP(Hyper Text Transfer Portocol)

HTTP is an important Portocol that powers the World Wide Web.

Check the Wikipedia Article or The article on Khan Academy for HTTP.

When is HTTP used

It is used when your browser sends a HTTP request to a Web Server to view the Website, and when the Server responds with a HTTP respond with the HTML and other attachments(ex. Images, CSS, JavaScript etc.) of the Website.

What is a Web Server

A Web Server is a Computer connected to the Internet to receive HTTP request from a Web Browser. It hosts Websites.

What is a Web Browser

A Web Browser is an Application that sends HTTP Requests from a Web Server, and render HTML, and other content found in a Webpage.

There are millions of Websites in the World Wide Web, but we want to visit only one. The IP(Internet Portocol) to the rescue, however, an IP address is not human friendly. What is DNS(Domain Name System)

The Domain Name System is the connection between humans or computers on addressing computers in the Internet. A human cannot easily remember an IP Address, like this (176.34.131.233), but a human can easily remember a Domain Name like this (duckduckgo.com).
